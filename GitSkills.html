Task 5: Learning to push your changes to github in playground repo

  1. Here you will now make some changes and push them in git repo.
    You should push your changes before you finish your tasks for the day. It would be good to push everyday once.

  2. Open the playground repo that you created in your earlier task in VSCode

    You will get a pop up window as shown below, just go to your folder and press selectFolder

  3. Now you can see a file index.html in your left window. Double click on it to open the index.html and see the content there

  4. Currently it is showing 'Hello my name is ParaGeek' , you have to change it to Hello my name is xyz and press 'ctrl+s' to save it.
  5. Now if you see the differences it will show you the changes and your task is to push the changes. See below figure for the steps

  6. Now you have to commit your changes before pushing.

  7. Now your 'git status' should show that you have done 1 commit. Just push your changes.

  8. Now go to your playground repo(https://github.com/parageek/PlayGroundRepo) and check if your changes appear .One way to verify this is check if the comments are the same as you had given

    If you see the changes then your task is completed! Congratulations!
